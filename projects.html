Projects
Contribution to MuMMER European Project
This project aims at developing a Multi Model Mall Entertainment Robot, interacting with humans and also guiding them to the desired locations in a Mall. As a part of LAAS team, I worked on Human Aware navigation and planning.

Motion Transfer from Human to Humanoid
This project aims at transferring simple human motions onto a humanoid robot. Human motions are captured using 8 Bonita Vicon cameras and the trajectories are stored. These trajectories are then carefully mapped to a humanoid robot using analytical Inverse Kinematics based planning.

Path planning and collision avoidance
Devised and implemented a methodology for path planning and collision avoidance of a differential drive wheeled robot for static as well as dynamic obstacles using RRT and velocity cones. This methodology takes future steps also into consideration while planning, unlike, traditional RRT based planners where only the present state is considered.

Finger print recognition using MKL-SVM
A method for finger print recognition is developed using Multi Kernel Learning Support Vector Machine as the base learner and different image processing techniques for feature extraction. The feature vector consisted of feature points called 'minutiae', the dimension which is variable. These dimensionality issues are addressed by proposing a constant vector representation using the given feature vector.

Text to emotive speech synthesis
This project involved implemented an end to end text to speech synthesizer with a choice for emotion of the systhesized speech. Text to speech synthesis system is developed using Festival framework and CMU ARCTIC databases. System was then extended to synthesize speech in 5 different emotions by modifying different elements of speech using MATLAB.

1-D Gantry Crane
Designed a 1-D gantry crane from scratch and implemented automated pick and place of metallic objects. Object detection is done using an IR beam and an electro-magnet then picks up the object. The crane then takes the object to the desired position (one end of the crane) and drops it. The whole automation and control is done using an Arduino.
